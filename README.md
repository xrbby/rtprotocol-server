# RTProtocolServer

## Installation

### Maven

#### Repository
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

#### Dependency
```xml
<dependency>
    <groupId>com.github.xrbby</groupId>
    <artifactId>rtprotocol-server</artifactId>
    <version>1.0.0</version>
</dependency>
```
